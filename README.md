# Tic-Tac-Toe Game
<img width="1824" height="969" alt="image" src="https://github.com/user-attachments/assets/f50e077c-2b77-49a6-b60f-76090ea764db" />


A simple, interactive Tic-Tac-Toe web application built using HTML, CSS, and JavaScript. 

## Features
- Classic 3x3 grid gameplay.
- Interactive and responsive UI.
- Win detection and game reset functionality.
- Built with pure JavaScript (no external libraries).

## Technologies Used
- **HTML5:** For the structure of the game grid.
- **CSS3:** For styling and layout.
- **JavaScript (ES6):** For game logic and event handling.
